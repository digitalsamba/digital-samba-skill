# Digital Samba SDK Reference

> This file will contain the complete SDK documentation.

## Installation

```bash
npm install @digitalsamba/embedded-sdk
```

Or via CDN:
```html
<script crossorigin src="https://unpkg.com/@digitalsamba/embedded-sdk"></script>
```
